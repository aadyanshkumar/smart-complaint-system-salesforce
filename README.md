# Smart Complaint Management System

A Salesforce-based complaint management application built using:

- Apex
- SOQL
- Triggers
- Async Apex
- Validation Rules
- Salesforce Admin Features

## Features

- Auto priority assignment
- Client complaint tracking
- Complaint escalation logic
- Delete restrictions
- Aggregate SOQL processing

## Tech Stack

- Salesforce Platform
- Apex
- SOQL
- VS Code
- Salesforce CLI
- Git & GitHub
